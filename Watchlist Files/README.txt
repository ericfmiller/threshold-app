Place your Seeking Alpha watchlist export files (Z-files) here.

These are .xlsx files exported from Seeking Alpha for asset watchlists.
Each file should contain 4 sheets: Summary, Ratings, Holdings, Dividends.

Expected naming convention:
  Z -Category Name YYYY-MM-DD.xlsx

Examples:
  Z -Stocks and EFTS to Consider as Investments 2026-02-16.xlsx
  Z -Energy Assets to Consider 2026-02-16.xlsx
  Z -REITs 2026-02-16.xlsx

Tickers found in these files will be auto-onboarded and scored
alongside your portfolio holdings.

IMPORTANT: These files contain personal financial data and are
excluded from git via .gitignore. They will NOT be committed.
